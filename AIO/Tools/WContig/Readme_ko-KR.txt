WinContig는 컴퓨터에 어떠한 설치 디렉토리 또는 레지스트리 항목도
만들지 않는, 사용하기 쉬운 독립 실행형 조각모음 도구입니다.
목적은 전체 디스크를 조각모음할 필요 없이 파일을 빠르게
조각모음하는 데 있습니다.
또한, WinContig는 프로파일 안에 그룹 파일을 허용합니다,
그리고 프로그램의 작동 방식을 제어하는데 사용할 수 있는 선택적인
명령줄 스위치를 다수 제공합니다.

WinContig는 Windows XP, Windows Vista, Windows 7, Windows 8/8.1/10
운영체제 용으로 설계된 응용 프로그램입니다.

Windows XP, Windows Vista, Windows 7 또는 Windows 8/8.1/10 사용자는
전체 관리자 권한으로 로그인해야 합니다.

====================================================================
최종 사용자 사용권 계약
====================================================================
이 소프트웨어와 모든 포함된 파일은 "있는 그대로" 제공되고 있으며,
특별한 용도를 위한 상업적이고도 합리적인 암묵적인 보증으로 한정하지
않고, 그것을 포함하고 있는 다음에 명시하고 있는 보증도 암시하고 있는
보증도 없습니다.
어떠한 경우라도, 제작자는 이 프로그램을 사용한 것 혹은 이 제품으로
할 수 없는 것에 따라 일어난 모든 손해(사업 이익의 손실, 사업의 중단,
업무상의 정보의 손실 혹은 다른 금전적 손실 등 모든 손해를 포함)에
대해서 책임지지 않습니다.

이 문서의 정보는 예고없이 변경될 수 있고,
제작자와의 약속을 나타내지는 않습니다.
이 문서에 설명된 소프트웨어는 이 사용권의 계약하에 제공됩니다.

이 약관에 동의하며 사용 목적에 맞게 해당 내용물을 사용할 것을
수락한다면, 언제든지 이 제품을 사용하세요.

====================================================================
설치
====================================================================
WinContig는 독립 실행형 실행 파일이며, 어떠한 설치 과정이나
추가 DLL이 필요치 않습니다.
사용하기 위해서는, 단지 이것을 다운로드해서 원하는 폴더에
ZIP 압축 파일을 풀고 실행하면 됩니다.

참고: WinContig의 최신 버전을 설치하려면 먼저 이전 버전을 제거해야
합니다. 또한, 처음 실행하면, 모든 설정은 기본값으로 설정됩니다.
