WinContig é uma ferramenta de desfragmentação stand-alone fácil de usar que não cria qualquer diretório de instalação ou entradas no Registro no seu computador. Sua finalidade é a desfragmentação rápida de arquivos sem a necessidade de desfragmentar o disco inteiro. Além disso, WinContig permite agrupar arquivos em perfis, e também aceita um grande número de opções em linha de comando opcionais que você pode usar para controlar a forma como utiliza o programa.

WinContig é uma aplicação designada para os sistemas operacionais Windows XP, Windows Vista, Windows Seven, Windows 8/8.1, Windows 10. 

Como um usuário do Windows XP, Windows Vista, Windows Seven, Windows 8/8.1 ou Windows 10 você deve estar logado em uma conta com todos os direitos de administrador.
